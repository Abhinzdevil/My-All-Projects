# Prime Opening Page Clone

This project is a clone of the Amazon Prime Video opening page UI.
It is built using HTML, CSS, and JavaScript to practice frontend
layout, styling, and basic interactivity.

## Features
- Prime Video inspired landing page UI
- Clean and responsive layout
- Header, banner, and content sections
- Simple and user-friendly design

## Technologies Used
- HTML5
- CSS3
- JavaScript (basic)

## Purpose
This project was created to improve frontend development skills,
especially UI cloning, layout design, and CSS styling.
