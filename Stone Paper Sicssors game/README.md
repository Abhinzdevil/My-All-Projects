# Stone Paper Scissors Game

A simple Stone Paper Scissors game built using HTML, CSS, and
JavaScript. This game allows the user to play against the computer
with random computer choices.

## Features
- User vs Computer gameplay
- Random computer move generation
- Score tracking
- Simple and interactive UI

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## Purpose
This project was built to practice JavaScript basics, logic building,
and event handling.
