# Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS,
and JavaScript. This project allows two players to play the game
on the same device.

## Features
- Two-player gameplay
- Interactive UI
- Win and draw detection
- Game reset functionality

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## Purpose
This project was created to practice JavaScript logic, conditional
statements, and DOM manipulation.
